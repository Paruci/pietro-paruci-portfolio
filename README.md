# Portfólio Pietro Paruci

Portfólio de desenvolvedor focado em UX/UI moderno e profissional com tema escuro.

## Descrição

Site único (single page) que apresenta dados de perfil profissional, formação, habilidades técnicas e comportamentais, experiência, idiomas e projetos.

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome (ícones)

## Estrutura do projeto

- `index.html` — página principal
- `styles.css` — estilos visuais e responsividade
- `script.js` — animações de rolagem, navbar e efeitos
- `package.json` — scripts de desenvolvimento
- `.gitignore` — arquivos e pastas ignorados

## Como rodar localmente

1. Instale o Node.js (LTS): https://nodejs.org/
2. No diretório do projeto, rode:

```bash
npm install
npm run dev
```

3. Abra `http://localhost:3000`

## Build

Não há etapa de build para versão estática; o comando abaixo é placeholder (sem ação efetiva):

```bash
npm run build
```
